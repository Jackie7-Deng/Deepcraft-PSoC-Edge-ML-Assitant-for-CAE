# 专属助手用户指南

**用途**: 说明如何与当前专属助手高效协作
**适用对象**: 开发者、CAE 工程师、PSoC Edge + DEEPCRAFT 使用者
**版本**: 1.2
**最后更新**: 2026-03-10

## 1. 当前助手能做什么

当前助手承担三类职责：
- **导师**：解释工作流与核心概念
- **图书管理员**：定位本地文档、示例与入口文件
- **技术支持**：排障并生成基于证据的回复草稿

## 2. 引用约定

推荐的本地引用形式：
- `docs/trm_aum/psoc_edge_e8x_architecture_rm.pdf p.79`
- `docs/readmes/README_Machine_Learning_Face_ID_Demo.md`
- `Machine_Learning_Imagimob_MTBML_Deploy/source/inference.c:L20-L48`
- `ingest/html/code_gen_infineon_boards.md`
- `knowledge/troubleshooting_playbook.md 2`

若没有直接证据，助手必须明确写出 `未在当前知识库找到依据`。

## 3. 按需求选择入口

- 总体定向：`knowledge/document_map.md`
- 工作流学习：`knowledge/ml_workflow.md`
- 故障排查：`knowledge/troubleshooting_playbook.md`
- 示例选择：`knowledge/examples_catalog.md`
- 参数 / 上下文补全：`knowledge/parameters_catalog.md`

## 4. 语言要求

- 助手默认使用中文回答
- 后续新增维护类文档默认使用中文编写
- 产品名、函数名、路径名可保留原始英文
