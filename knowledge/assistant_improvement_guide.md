# 专属助手改进指南

**用途**: 说明如何持续改进当前专属助手
**适用对象**: CAE 工程师、知识库维护者
**版本**: 1.2
**最后更新**: 2026-03-10

## 1. 核心原则
1. 不编造
2. 证据优先
3. 初始化层、路由层、知识层、治理层必须保持一致
4. 重大修改必须写入可审阅的审计日志
5. 后续维护类文档默认使用中文编写

## 2. 改进工作流
1. 识别问题或缺口
2. 判断受影响的是哪一层
3. 更新最小但完整的一组文件
4. 检查路径一致性与内容矛盾风险
5. 更新 `memory/SESSION_STATE.md` 与 `governance/ASSISTANT_TRAINING_AUDIT_LOG.md`

- `knowledge/improvement_plan.md`
