# Install Download Studio

### Download and Install DEEPCRAFT™ Studio

This section provides the information to download and install DEEPCRAFT™ Studio.

To download and Install Studio, follow the steps:

1. - Navigate to [Imagimob Sign up page](https://account.imagimob.com/signup).

  ![](https://developer.imagimob.com/Images/dc-5.png)
2. - Enter your email address and password.
3. - Re-enter your password to confirm.
4. - Read and accept the [DEEPCRAFT™ Studio Terms & Conditions](https://developer.imagimob.com/legal/imagimob-studio-terms-and-conditions) and click **Sign up**.
  A confirmation email is sent to your email address. Confirm your email address to complete the verification.
5. - Click **Download** to download the DEEPCRAFT™ Studio executable file.

  ![](https://developer.imagimob.com/Images/dc-4.png)
6. - Locate and run the DEEPCRAFT™ Studio executable file. The DEEPCRAFT™ Studio Setup wizard appears.

Make sure you do not run the executable file as an administrator, as this might cause certain issues when using the DEEPCRAFT™ Studio.

1. - Read the terms of license agreement. If you agree, select **I accept the agreement** and click **Next**.
2. - Follow the prompts to complete the installation. Once the installation is successful, the DEEPCRAFT™ Studio login screen appears.

  ![](https://developer.imagimob.com/Images/install-download/pic-1.png)
3. - Enter the email address and password and click **Log In**. The DEEPCRAFT™ Studio opens up.

  ![](https://developer.imagimob.com/Images/deepcraft.png)

You can also install DEEPCRAFT™ Studio from [Infineon Developer Center Launcher](https://www.infineon.com/cms/en/design-support/tools/utilities/infineon-developer-center-idc-launcher/?redirId=102702) and [ModusToolbox™ Setup](https://softwaretools.infineon.com/tools/com.ifx.tb.tool.modustoolboxsetup).
