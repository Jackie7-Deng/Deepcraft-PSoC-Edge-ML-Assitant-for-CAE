### Model Deployment on supported Infineon Boards

This section specifically addresses deploying the model on Infineon boards using ModusToolbox™ or Aurix™ Development Studio. To simplify the deployment process, we provide a code example that includes a pre-generated model, ready for deployment on Infineon boards.

Once you have built your own model, you can replace the model in the code example with your own model created in DEEPCRAFT™ Studio. This allows you to use the example as-is to deploy your model on Infineon boards.

To learn how to deploy a trained model onto the Infineon board, using ModusToolbox™ or AURIX™ Development Studio, refer to the respective sections based on the type of board you are using.
