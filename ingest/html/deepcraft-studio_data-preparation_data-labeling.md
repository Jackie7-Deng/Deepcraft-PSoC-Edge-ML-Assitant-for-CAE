### Data Labeling

Data labeling is an integral step in developing high performing machine learning models. It is the process of converting unstructured data to structured data by adding informative labels, that helps the machine learning model to make accurate predictions. Studio allows you to annotate unstructured data, like audio and time series, and develop structured data to create datasets for training the machine learning models. Studio also supports uploading a video that can be used as visual reference for annotating the time series data.

#### Data Labeling approaches in Studio

There are different ways for labeling data in Studio:

- - [Manual Labeling](https://developer.imagimob.com/data-preparation/data-labeling/manual-labeling)
- - [ML Assisted labeling](https://developer.imagimob.com/data-preparation/data-labeling/ml-assisted-labeling)
- - [Sequential Labeling](https://developer.imagimob.com/data-preparation/data-labeling/add-label)
