# Multiple Tracks

### Creating multiple label tracks

You can create multiple label tracks in a session file.

To create the label track, follow the steps:

1. - Double-click the session file (.imsession) in which you want to add a label track. The session file opens in a new tab.

  ![create-label](https://developer.imagimob.com/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Flabel-create.07c9c31d.png&w=1920&q=75)
2. - Click **Add Track** to add a label track in the session file. The **New Track** window appears.

  ![select-label](https://developer.imagimob.com/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Flabel-select.79293225.png&w=1920&q=75)
3. - On the left pane, select **Label Tracks > Label Track** from the list of options and click **OK**. The **Add New Label Track** window appears.

  ![](https://developer.imagimob.com/Images/label-box.png)
  * In **Track Name**, enter the name of the track.
  * In **Payload File**, the name of the label file (track name) is displayed automatically.
4. - Click **Add** to add the label track.

  ![](https://developer.imagimob.com/Images/label-new.png)

  A new label track is added in the session(.imsession) file.
