# Model Deployment Siren Tutorial

### Code Generation and Model Deployment

The last step is to generate the code and deploy the model on the board. We will generate the code for the model using Studio, and deploy the model on the Infineon board using ModusToolbox™ or Aurix™ Development Studio.

To generate and optimize the code on Infineon PSOC™ 6 and PSOC™ Edge boards, refer to [Code Generation and Model Optimization for Infineon PSOC™ 6 and PSOC™ Edge boards](https://developer.imagimob.com/code-generation/code-gen-infineon-boards) for detailed instructions.

To generate the code for Infineon AURIX™ TC375, TRAVEO™ T2G boards and XMC7200 Evaluation Kit, refer to [Code Generation for Infineon AURIX, Traveo, XMC7200 and Other Development Board](https://developer.imagimob.com/code-generation/code-gen-aurix-other-dev-boards#code-generation-for-infineon-aurix-and-other-development-boards).

To deploy the siren detection machine learning model, refer to the respective sections based on the type of board you are using.

- - [Deploy siren detection model on PSOC™ 6 and PSOC™ Edge boards](https://developer.imagimob.com/deployment/deploy-models-supported-boards/deploy-siren-detection-model-PSoC-boards)
- - [Deploy siren detection model on TRAVEO™ T2G (KIT\_T2G-B-E\_LITE)](https://developer.imagimob.com/deployment/deploy-models-supported-boards/deploy-siren-detection-model-T2G-B-E-lite)
- - [Deploy siren detection model on TRAVEO™ T2G (KIT\_T2G-B-H\_LITE)](https://developer.imagimob.com/deployment/deploy-models-supported-boards/deploy-siren-detection-model-T2G-B-H-lite)
- - [Deploy siren detection model on AURIX™ TC375 Lite Kit](https://developer.imagimob.com/deployment/deploy-models-supported-boards/deploy-siren-detection-model-aurix-TC375-lite)
- - [Deploy siren detection model on XMC7200 Evaluation Kit (KIT\_XMC72\_EVK)](https://developer.imagimob.com/deployment/deploy-models-supported-boards/deploy-siren-detection-model-on-XMC7200)
